# 安装
    ```
    # 克隆项目
    git clone https://github.com/scory02/scory.git

    # 进入项目目录
    cd scory

    # 安装依赖
    npm install

    # 建议不要用 cnpm 安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
    npm install --registry=https://registry.npmmirror.com

    # 本地开发 启动项目
    npm run dev
    
    ```
